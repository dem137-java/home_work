Домашнее задание реализовано в следующих классах:
DataContainer - основной класс-контейнер;
DataContainerTest - класс для тестирования;
IntegerComporator - класс с реализацией сравнения для значений Integer;
StringComporator - класс с реализацией сравнения для значений String;
справочно: класс DataContainerTest использовался для тестирования методов контейнера

Также в ДЗ содержится пакет taskStudent с реализацией задания, полученного во время занятия для самостоятельной
проработки, он содержит:
1) пакет comparators c реализацией сортировки по возрасту (AgeComporator), оценке (MarkComporator) и имени (NameComporator);
2) класс Student - класс, предназначенный для создания объектов типа Student
3) класс StudentMain - класс с точкой входа программу, реализующий создание объектов типа Student со случайным именем
4) класс StudentMain2RusNames - класс-аналог(копия) StudentMain, но реализующий создание объектов типа Student с корректным именем
